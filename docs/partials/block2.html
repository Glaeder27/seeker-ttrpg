TL;DR — Stances & Initiative

▸
  • Stances determine your role, tags, and initiative each round.
  • At the start of every round, players choose their stance secretly and reveal it simultaneously.
  • The GM then calls stances in a fixed order (e.g., Aggressive → Evasive → Balanced → ...).
  • Each stance defines which Tags you may use and how many Actions you receive.
  • Actions are spent to attack, defend, maneuver, or escalate during conflicts.
  • All characters start with access to six Basic Stances.
  • Advanced stances (e.g., Ambush, Berserk) are unlocked via Skills and Masteries.
  • When multiple actors share a stance, turn order can be resolved by table-preferred methods (e.g., Speed stat, GM call, clockwise).
