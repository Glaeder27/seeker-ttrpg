TL;DR — Tag System & Functionality

▸
  • Tags are keywords that define how effects interact mechanically.
  • They apply to Skills, Items, Masteries, Afflictions, and more.
  • Each tag has a clear, specific meaning (e.g., Attack, Fire, Movement).
  • Tags enable synergies, conditional effects, and combo logic.
  • Designers and players can reference tags instead of rewriting rules.
  • The Tag Glossary lists all defined tags, grouped by category.
